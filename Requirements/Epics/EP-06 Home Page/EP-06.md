# EP-06 — Home Page

## Goal
Provide a role-adaptive home page that surfaces the most relevant content and actions for each user type — from public visitors to sellers and admins — without requiring navigation.

## Role-Based Content

| Section | Visitor | Buyer | Seller | Admin |
|---|---|---|---|---|
| Event countdown | ✓ | ✓ | ✓ | — |
| Global donation progress bar | ✓ | ✓ | ✓ | — |
| Per-location donation breakdown | ✓ | ✓ | ✓ | — |
| Top donors leaderboard | ✓ | ✓ | ✓ | — |
| Sign Up / Log In CTA | ✓ | — | — | — |
| Featured shops (quick browse) | — | ✓ | ✓ | — |
| Basket summary (item count) | — | ✓ | — | — |
| My shop status + recent orders | — | — | ✓ | — |
| Redirects to Admin Dashboard | — | — | — | ✓ |

## In Scope
- Public landing content (donation progress, countdown, leaderboard) — no login required
- Logged-in buyer home (featured shops, basket summary, donation CTA)
- Seller home (shop status: open/closed, orders received count, link to shop dashboard)
- Admin auto-redirect to Admin Dashboard (EP-02)
- Adaptive layout — same URL, content changes based on auth state

## Out of Scope
- Personalised product recommendations (TBD)
- Notification feed / activity stream (TBD)

## Roles Involved
- Visitor (unauthenticated)
- Logged-in user / Buyer
- Seller
- Admin (redirected to dashboard)

## User Stories

| # | Priority | Story |
|---|---|---|
| US-06.01 | High | As a **visitor**, I want to see the event's donation progress, per-location breakdown, and countdown on the home page, so that I can quickly understand the event and decide to participate |
| US-06.02 | High | As a **logged-in user**, I want to see featured shops and my basket summary on the home page, so that I can quickly continue browsing or check out |
| US-06.03 | Medium | As a **Seller**, I want to see my shop's current status and recent order count on the home page, so that I can monitor my shop at a glance without navigating away |
