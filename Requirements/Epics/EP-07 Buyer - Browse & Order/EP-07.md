# EP-07 — Buyer - Browse & Order

## Goal
Allow buyers to discover products across all shops, place orders, and track them through to fulfilment.

## In Scope
- Browse all shops
- View products within a shop (name, description, price, photos)
- Filter / search products by category (artisan goods, food)
- View product detail page
- Add products to a cart / order
- Place an order (confirm items, quantity, pick-up details)
- View order confirmation
- Track order status (pending, ready for pick-up, completed)
- View order history
- Seller: update order status

## Out of Scope
- Real payment processing — payment mechanism TBD
- Delivery / shipping (pick-up only — TBD)
- Order cancellation flow (TBD)
- Product reviews or ratings (TBD)

## Roles Involved
- Employee (Buyer)
- Employee (Seller)
