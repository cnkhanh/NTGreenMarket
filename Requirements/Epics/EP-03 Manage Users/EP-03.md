# EP-03 — Manage Users

## Goal
Allow Admins to assign event-scoped Seller roles to registered users and manage participant access.

## In Scope
- Assign Seller role to a registered user for a specific event
- Revoke or change a user's role
- Deactivate a user's account
- View all users and their current roles
- Seller role automatically becomes read-only when event ends
- Admin promotion: see EP-01 US-01.05

## Out of Scope
- Admin-created accounts (users self-register)
- Bulk import of users (TBD)

## Roles Involved
- Admin

## User Stories

| # | Priority | Story |
|---|---|---|
| US-03.01 | High | As an **Admin**, I want to assign the Seller role to a registered user for a specific event, so that they can set up a shop |
| US-03.02 | Medium | As an **Admin**, I want to revoke or change a user's role, so that I can control who has seller or admin access |
| US-03.03 | Medium | As an **Admin**, I want to deactivate a user's account, so that I can remove inactive or inappropriate participants |
| US-03.04 | Medium | As an **Admin**, I want to view all users and their current roles, so that I have full visibility of who is participating |
