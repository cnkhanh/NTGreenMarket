# EP-01 — User Authentication & Access

## Goal
Enable employees to securely log in and access the platform with the appropriate role (Buyer, Seller, or Admin).

## In Scope
- Employee login (SSO or credential-based — TBD)
- Role assignment: Buyer, Seller, Admin
- Session management and logout
- Access control — restrict features by role

## Out of Scope
- Self-registration (accounts provisioned by Admin or via SSO — TBD)
- Password reset flow (if using SSO)

## Roles Involved
- Employee (Buyer)
- Employee (Seller)
- Admin
