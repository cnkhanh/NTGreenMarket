# EP-07 — Browse & Order

## Goal
Allow logged-in users to discover shops and products across all locations, add items to a basket, and place orders for pick-up — restricted to their own location's shops.

## Location Rules
- Logged-in users can **browse** shops from all 3 locations (Ha Noi, Da Nang, Ho Chi Minh)
- Users can only **add to basket and order** from shops in their own registered location
- When viewing a product from a different location, show: *"You can only order from [user's location] shops. This shop is in [shop's location]."*
- Basket only contains items from the user's own location

## Access
- Login required for all features in this epic (browse, basket, order, track)
- Public visitors can only access EP-04 (Donations) and EP-08 (Timeline & Countdown)

## In Scope
- Browse all shops across all locations
- Filter / search shops and products by category and location
- View product detail page
- Add products to basket (own location only)
- View and manage basket (update quantities, remove items)
- Place order from basket (confirm items, pick-up details)
- View order confirmation
- Track order status (pending, ready for pick-up, completed)
- Seller: update order status

## Out of Scope
- Cross-location ordering
- Real payment processing — pick-up only
- Delivery / shipping
- Order cancellation flow (TBD)
- Product reviews or ratings (TBD)

## Roles Involved
- Logged-in user / Buyer
- Seller

## User Stories

| # | Priority | Story |
|---|---|---|
| US-07.01 | High | As a **logged-in user**, I want to browse shops across all locations, so that I can discover what's available at the market |
| US-07.02 | Medium | As a **logged-in user**, I want to filter shops and products by category and location, so that I can find what I'm looking for quickly |
| US-07.03 | High | As a **logged-in user**, I want to add products from shops in my location to my basket, so that I can collect items before placing an order |
| US-07.04 | High | As a **logged-in user**, I want to view and manage my basket, so that I can review items before checking out |
| US-07.05 | High | As a **logged-in user**, I want to place an order from my basket, so that I can secure my items for pick-up day |
| US-07.06 | High | As a **logged-in user**, I want to view my order confirmation, so that I know my order was successfully placed |
