# EP-07 — Browse & Order

## Goal
Allow buyers to discover products across all shops, place orders, and track them through to fulfilment.

## In Scope
- Browse all shops
- View products within a shop (name, description, price, photos)
- Filter / search products by category (artisan goods, food)
- View product detail page
- Add products to a cart / order
- Place an order (confirm items, quantity, pick-up details)
- View order confirmation
- Track order status (pending, ready for pick-up, completed)
- View order history
- Seller: update order status

## Out of Scope
- Real payment processing — payment mechanism TBD
- Delivery / shipping (pick-up only — TBD)
- Order cancellation flow (TBD)
- Product reviews or ratings (TBD)

## Access
- Login required for all features in this epic (browse, search, order, track)
- Public visitors can only access EP-04 (Donations) and EP-08 (Timeline & Countdown)

## Roles Involved
- Logged-in user / Buyer
- Seller

## User Stories

| # | Priority | Story |
|---|---|---|
| US-07.01 | High | As a **logged-in user**, I want to browse all shops, so that I can discover what's available at the market |
| US-07.02 | Medium | As a **logged-in user**, I want to filter and search products by category, so that I can find what I'm looking for quickly |
| US-07.03 | High | As a **logged-in user**, I want to place an order from a shop, so that I can secure my items before market day |
| US-07.04 | High | As a **logged-in user**, I want to view my order confirmation, so that I know my order was successfully placed |
| US-07.05 | Medium | As a **logged-in user**, I want to track my order status, so that I know when my items are ready for pick-up |
| US-07.06 | High | As a **Seller**, I want to update the status of an order, so that buyers know when their items are ready |
