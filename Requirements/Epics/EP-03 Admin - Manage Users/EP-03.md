# EP-03 — Admin - Manage Users

## Goal
Allow Admins to provision and manage user accounts and event-scoped role assignments for each event.

## In Scope
- Create seller accounts for a specific event (Admin-only — no self-registration)
- Assign roles per event: Buyer (default for all employees), Seller, Admin
- Deactivate or remove a user's access for the current event
- View all users and their roles for the current event
- Seller accounts expire at the end of the event — access is automatically revoked

## Out of Scope
- Bulk import of users (TBD)
- Managing accounts outside of an active event
- Permanent admin accounts beyond the event lifecycle (TBD)
