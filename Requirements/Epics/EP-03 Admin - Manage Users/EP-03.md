# EP-03 — Admin - Manage Users

## Goal
Allow Admins to manage employee accounts and control role assignments on the platform.

## In Scope
- View all registered users
- Assign or change user roles (Buyer, Seller, Admin)
- Enable / disable user accounts
- View user activity summary

## Out of Scope
- User account creation (handled via SSO or provisioning — TBD)
- Bulk import of users (TBD)

## Roles Involved
- Admin
