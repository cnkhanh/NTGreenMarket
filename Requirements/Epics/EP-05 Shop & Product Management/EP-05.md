# EP-05 — Shop & Product Management

## Goal
Allow employees to set up and manage their own online shop and product listings within the marketplace.

## In Scope
- Create a shop (name, description, banner/logo)
- Edit shop details
- Open / close shop for orders
- Add, edit, and remove products (name, description, price, photos, category)
- View shop dashboard (orders received, revenue summary)
- Delete / deactivate shop

## Out of Scope
- Multiple shops per seller (one shop per seller — TBD)
- Inventory tracking / stock management (TBD)

## Roles Involved
- Employee (Seller)
- Admin (oversight)

## User Stories

| # | Priority | Story |
|---|---|---|
| US-05.01 | High | As a **Seller**, I want to create my shop with a name, description, and banner, so that buyers can discover who I am |
| US-05.02 | High | As a **Seller**, I want to add products with photos, price, category, and description, so that buyers know what I'm selling |
| US-05.03 | High | As a **Seller**, I want to edit and remove products, so that I can keep my shop up to date |
| US-05.04 | High | As a **Seller**, I want to open and close my shop for orders, so that I control when buyers can order |
| US-05.05 | High | As a **Seller**, I want to view orders I've received, so that I can prepare products for pick-up day |
| US-05.06 | Medium | As a **Seller**, I want to view my past shops in read-only mode after an event ends, so that I can reference my previous listings |
| US-05.07 | Medium | As a **Seller**, I want to re-open an old shop when assigned as Seller for a new event, so that I don't have to recreate everything from scratch |
