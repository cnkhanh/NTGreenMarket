# EP-02 — Manage Event

## Goal
Allow Admins to configure, launch, and oversee the NT Green Market event from start to finish — including a real-time dashboard showing event health across all locations.

## In Scope
- Create and configure the event (name, dates, donation target)
- Publish / unpublish the event
- Close the event and manage post-event state
- Admin dashboard: real-time overview of donations, orders, and shops across all locations
- Per-location breakdown on dashboard (Ha Noi / Da Nang / Ho Chi Minh)

## Out of Scope
- Financial reporting (TBD)
- Communication / notifications to employees (TBD)

## Roles Involved
- Admin

## User Stories

| # | Priority | Story |
|---|---|---|
| US-02.01 | High | As an **Admin**, I want to create and configure an event (name, dates, donation target), so that the platform is ready for the market |
| US-02.02 | High | As an **Admin**, I want to publish and unpublish the event, so that I control when it's accessible |
| US-02.03 | High | As an **Admin**, I want to view a dashboard showing total donations, orders, and active shops across all locations, so that I can monitor event health at a glance |
| US-02.04 | Medium | As an **Admin**, I want to see a per-location breakdown on the dashboard, so that I can compare engagement across offices |
| US-02.05 | High | As an **Admin**, I want to close the event, so that no new orders or donations are accepted after it ends |
