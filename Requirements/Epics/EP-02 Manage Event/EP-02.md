# EP-02 — Manage Event

## Goal
Allow Admins to configure, launch, and oversee the NT Green Market event from start to finish.

## In Scope
- Create and configure the event (name, dates, description)
- Publish / unpublish the event
- Monitor event activity (shops, orders, donations)
- Close the event and manage post-event state

## Out of Scope
- Financial reporting (TBD)
- Communication / notifications to employees (TBD)

## Roles Involved
- Admin

## User Stories

| # | Priority | Story |
|---|---|---|
| US-02.01 | High | As an **Admin**, I want to create and configure an event (name, dates, donation target), so that the platform is ready for the market |
| US-02.02 | High | As an **Admin**, I want to publish and unpublish the event, so that I control when it's accessible |
| US-02.03 | Medium | As an **Admin**, I want to monitor event activity (shops, orders, donations), so that I can track progress towards the charity goal |
| US-02.04 | High | As an **Admin**, I want to close the event, so that no new orders or donations are accepted after it ends |
