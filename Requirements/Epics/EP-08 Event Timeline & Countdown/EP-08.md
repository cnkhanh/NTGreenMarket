# EP-08 — Event Timeline & Countdown

## Goal
Display the event schedule and a live countdown to keep all participants informed and engaged throughout the event.

## In Scope
- Event countdown timer (days/hours remaining)
- Event timeline / key dates (open, close, pick-up day, etc.)
- Visible to all visitors (no login required)
- Admin: manage timeline entries

## Out of Scope
- Push notifications or reminders (TBD)
- Personalised schedules per user

## Roles Involved
- Employee (Buyer)
- Employee (Seller)
- Admin

## User Stories

| # | Priority | Story |
|---|---|---|
| US-08.01 | Medium | As **any visitor**, I want to see a live countdown to market day, so that I know how much time I have left to order |
| US-08.02 | Medium | As **any visitor**, I want to view the event timeline with key dates, so that I can plan my participation |
| US-08.03 | Medium | As an **Admin**, I want to manage timeline entries, so that I can keep the event schedule accurate |
