# EP-08 — Event Timeline & Countdown

## Goal
Display the event schedule and a live countdown to keep all participants informed and engaged throughout the event.

## In Scope
- Event countdown timer (days/hours remaining)
- Event timeline / key dates (open, close, pick-up day, etc.)
- Visible to all authenticated users
- Admin: manage timeline entries

## Out of Scope
- Push notifications or reminders (TBD)
- Personalised schedules per user

## Roles Involved
- Employee (Buyer)
- Employee (Seller)
- Admin
