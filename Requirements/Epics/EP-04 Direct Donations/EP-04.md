# EP-04 — Direct Donations

## Goal
Enable anyone (including visitors) to donate to the event's charity fund. The platform drives fundraising through a clear target, progress bars, and a donor leaderboard — shown both globally and broken down by location.

## Locations
- Ha Noi · Da Nang · Ho Chi Minh
- Donations are tracked per location for each event
- Global total and per-location totals are both visible to all visitors

## In Scope
- Donate to the event charity fund — no login required
- Progress bar showing total raised vs event donation target (global view)
- Per-location donation breakdown — progress bar and total per location (Ha Noi / Da Nang / Ho Chi Minh)
- Leaderboard of top donors (global)
- Admin: set and update the global donation target per event

## Out of Scope
- Per-location donation targets (single global target per event)
- Multiple charity selection (single fund per event)
- Donation history per user
- Recent donations feed
- Real payment processing — donation mechanism TBD (internal pledge, in-person cash, etc.)
- Donation receipts or tax documentation

## Design Notes
- Donation CTA should be prominent on the homepage and throughout the app
- Progress bar should feel energetic and celebratory as it fills
- Per-location breakdown encourages friendly inter-office competition
- Leaderboard drives social motivation — show display names (not full employee IDs)

## Roles Involved
- Any visitor (donate, view global progress, view per-location breakdown, view leaderboard)
- Admin (set global donation target)

## User Stories

| # | Priority | Story |
|---|---|---|
| US-04.01 | High | As **any visitor**, I want to donate to the event's charity fund, so that I can contribute to the fundraising goal |
| US-04.02 | High | As **any visitor**, I want to see a global progress bar showing total raised vs the event's donation target, so that I'm motivated to contribute |
| US-04.03 | High | As **any visitor**, I want to see a per-location donation breakdown, so that I can see how each office is contributing |
| US-04.04 | Medium | As **any visitor**, I want to see a leaderboard of top donors, so that I can see community generosity at a glance |
| US-04.05 | High | As an **Admin**, I want to set a donation target for the event, so that there's a clear fundraising goal to work towards |
