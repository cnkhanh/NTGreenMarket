# EP-04 — Direct Donations

## Goal
Enable anyone (including visitors) to donate to the event's charity fund. The platform drives fundraising through a clear target, a progress bar, and a donor leaderboard.

## In Scope
- Donate to the event charity fund — no login required
- Progress bar showing total raised vs event donation target
- Leaderboard of top donors
- Admin: set and update the donation target per event

## Out of Scope
- Multiple charity selection (single fund per event)
- Donation history per user
- Recent donations feed
- Real payment processing — donation mechanism TBD (internal pledge, in-person cash, etc.)
- Donation receipts or tax documentation

## Design Notes
- Donation CTA should be prominent on the homepage and throughout the app
- Progress bar should feel energetic and celebratory as it fills
- Leaderboard drives social motivation — show display names (not full employee IDs)

## Roles Involved
- Any visitor (donate, view progress, view leaderboard)
- Admin (set target)

## User Stories

| # | Priority | Story |
|---|---|---|
| US-04.01 | High | As **any visitor**, I want to donate to the event's charity fund, so that I can contribute to the fundraising goal |
| US-04.02 | High | As **any visitor**, I want to see a progress bar showing total raised vs the event's donation target, so that I'm motivated to contribute |
| US-04.03 | Medium | As **any visitor**, I want to see a leaderboard of top donors, so that I can see community generosity at a glance |
| US-04.04 | High | As an **Admin**, I want to set a donation target for the event, so that there's a clear fundraising goal to work towards |
